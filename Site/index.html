Desdemona

No fuss. Just the classic game of Reversi against an AI player.

Download Desdemona __VERSION__ (0.25 MB disk image). This is a Universal Binary, and should run on Mac OS X 10.3.9 and later.

You can subscribe to new releases via RSS.

The AI beats it creator most of the time at the default level. (But stronger and weaker settings are available.) No opening- or end-game database is used, so it is doubtful that it will perform well against strong players.

Desdemona defaults to using the classic 8x8 board but supports board sizes from 4x4 up to 20x20. Unlimited undo is available should you need it.
