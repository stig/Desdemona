Desdemona

No fuss. Just the classic game of Reversi against an AI player.

Download Desdemona @VERSION@ (0.25 MB disk image). This is a Universal Binary, and should run on Mac OS X 10.3.9 and later. New in this release:

@DESCRIPTION@

The AI beats it creator about 70% of the time at the default level, but stronger and weaker settings are available. However, no opening- or end-game database is used, so it is doubtful that it will perform well against really strong players.

Defaults to the classic 8x8 board, but supports board sizes from 4x4 up to 20x20. Unlimited undo is available should you need it.
