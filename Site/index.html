Desdemona

No fuss. Just the classic game of Reversi against an AI player.

Download Desdemona @VERSION@ (0.25 MB disk image). This is a Universal Binary, and should run on Mac OS X 10.3.9 and later.

Defaults to the classic 8x8 board, but also supports board sizes from 4x4 up to 20x20. Multiple difficulty levels are also supported.

In creating this game I borrowed some ideas and high-quality graphics from Gnome Iagno.

Desdemona is written and maintained by Stig Brautaset. This program is Free Software, and released under the GPL. You can get the source from my Subversion repository. If you want to hack on Desdemona you also need SBAlphaBeta.
